# Dashboard Project Summary & Sharing Guide

This package contains the "Support Command Center" dashboard project and representative screenshots of the three support tiers.

## 📊 Dashboard Tiers

- **Tier 1 (Frontline):** General support metrics, queue status, and volume tracking.
- **Tier 2 (Technical):** Specialized technical support metrics with dedicated team leaderboard.
- **Tier 3 (Development):** High-level engineering escalation tracking and pending fixes.

## 📁 What's in this folder?

1. **`dashboard_project.zip`**: The full source code (Next.js 16).
2. **`Tier1_Frontline.png`**: Screenshot of the Frontline view.
3. **`Tier2_Technical.png`**: Screenshot of the Technical view.
4. **`Tier3_Development.png`**: Screenshot of the Development view.

## 🚀 How to Run (for the team)

1. Unzip `dashboard_project.zip`.
2. Ensure you have Node.js installed.
3. Run `npm install` (this will take a few minutes).
4. Rename `.env.example` to `.env.local` and add your `VITE_INTERCOM_TOKEN`.
5. Run `npm run dev`.
6. Access at `http://localhost:3000`.

## ☁️ Uploading to Google Drive

1. Open your Google Drive folder.
2. Drag and drop this entire `Team_Presentation` folder into the Drive.
3. Sharing permissions: Ensure anyone with the link can view.
